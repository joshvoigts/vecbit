Bool as a service

Set or unset a vector of 63 bits by API.

Why?

A silly side project for giving everyone in the world a free bool on my cheap $5 VPS. Also, as an excuse for learning API design and authentication with stateless magic links. See code on github.

Quick start

Note that when using the code samples included in this documentation, requests should be sent with a Content-Type of application/json.

Authentication

Use of the API requires an API access token. To request this token, go to Settings in the menu to the left and "Create new access token".

To make calls to the API, send the API token in the Authorization HTTP header using the Bearer authentication scheme.

curl -H "Authorization: Bearer aTwSv..." https://vecbit/api/v1/bools
